03. Some Things I’ve Built

Suggesto

Suggesto is a one stop for movie suggestions and recommendations. Everyday one unique movie is suggested and users can also login, wishlist movies, follow their friends and share movies with them. We also show where the movie is streamed with direct OTT links, just one click watch on their repective OTT platforms. There are lot of features that are packed in it. Click below to check them out. This is built using Angular, Bootstrap and Firebase.

Webapp for Youtuber : RiseAloneYt

This is a supporting website for a Youtuber : RiseAloneYt. This is built using Angular, YoutubeAPI to retrieve channel and video details and Firebase for database, authentication and hosting.

Sg Photographi

This is a demo project I've built while learning Django and python. Modules involve authentication, searching, booking order, wallpapers, news feed and admin acess. This is built using Django as backend and HTML, CSS and JavaScript.

CRUD using Angular and .net Mvc and WebApi

This is a learning project developed while learning Angular and Asp.Net MVC. This project involve all the Create, read, update and delete operations on data from angular frontend to .net backend using Rest APIs.

Multiple Instagram Bio

This is a utility app to overcome the problem users face in instagram. Instagram allows users to have only one bio link, but users do have multiple bio links. So We provide a wall for each user in which users can add and delete any number of bios they want.

onGoing project

WeatherAtt
